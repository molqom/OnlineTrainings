Пакеты используются для упорядочивания класса по логическому, функциональному принципу. Именуются пакуты именем существительным маленькими буквами в единственном числе. По соглашению имя пакета должно совпадать с доменным именем организации. Если в доменном имени содержится недопустимы символ, то по соглашению допускается использовать нижнее подчеркивание
